## 🚀 如何使用

### 方法1：直接打开（最简单）

1. 双击打开 `index.html` 文件
2. 网站将在您的默认浏览器中打开

### 方法2：使用本地服务器（推荐）

#### 使用Python：
```bash
# 在项目文件夹中打开命令行/PowerShell
python -m http.server 8000
# 然后访问 http://localhost:8000
```

#### 使用Node.js：
```bash
npx http-server
# 然后访问显示的地址（通常是 http://localhost:8080）
```

#### 使用VS Code：
1. 安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🎨 设计说明

## 🔧 如何自定义

### 添加新文章

编辑 `data/articles.json` 文件，添加新的文章对象：

```json
{
  "id": 7,
  "title": "文章标题",
  "slug": "article-slug",
  "category": "fashion",
  "date": "2025-09-15",
  "author": "作者名字",
  "image": "图片URL",
  "excerpt": "文章摘要...",
  "content": "<p>完整HTML内容...</p>",
  "products": [...]
}
```

### 修改颜色

编辑 `css/styles.css` 中的CSS变量：

```css
:root {
  --primary-color: #E8D5D3;
  --accent-color: #C4A29E;
  /* 其他颜色... */
}
```

### 更换Logo

在HTML文件的header部分找到SVG代码并修改。

### 修改联系信息

在所有HTML文件的footer部分更新联系信息。

## 🎨 设计理念

网站采用"柔和/甜美"风格设计：

- **低饱和度色彩**: 使用莫兰迪色系，避免刺眼的鲜艳色彩
- **圆润边角**: 所有元素都有圆角，传达温暖友好感觉
- **细腻动画**: 悬停、点击时有平滑的过渡效果
- **留白设计**: 充足的空间让内容呼吸，不拥挤
- **优雅字体**: 使用Playfair Display（标题）和Poppins（正文）
