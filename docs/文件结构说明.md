## 📝 文件命名规范

### HTML文件
- 小写字母 + 连字符
- 例: `index.html`, `article.html`

### CSS文件
- 小写字母 + 连字符
- 例: `styles.css`, `article.css`

### JavaScript文件
- 小写字母 + 连字符
- 例: `main.js`, `articles-data.js`

### 文档文件（中文）
- 中文 + `.md` 扩展名
- 例: `使用说明.md`, `优化实施报告.md`

## 🔄 文件维护建议

### 定期更新
- **sitemap.xml**: 添加新文章时更新
- **更新日志.md**: 每次重大更新时记录
- **优化报告**: 每月或重大优化后更新

### 归档策略
- 旧版本报告可移至 `docs/reports/archive/`
- 保留最近3个月的详细报告
- 重要的历史报告永久保存

### 文档同步
- `README.md` 与 `使用说明.md` 保持内容一致
- 英文文档与中文文档同步更新

## 🎨 项目特点

### 设计风格
- 🎨 **Morandi配色**: 低饱和度、温暖舒适
- 🔘 **圆角设计**: 友好亲切的视觉效果
- ✨ **平滑动画**: 流畅的用户体验

### 技术特点
- 🚀 **纯静态网站**: 无需后端服务器
- 📱 **移动端响应**: 完美适配各种设备
- 🔍 **SEO优化**: 完整的Meta标签和sitemap
- ⚡ **性能优化**: JS异步加载、优化渲染
